# Online-Store-Data-Analysis(Intractive dashboard creation using MS excel)
## project objectives
 the store wants to create an anual sales report for the analyzation of the performance of sales and all other components.
 the owner of the store can understand their customers and grow more sales in next year.

 ## Dataset used 
 -<a href="https://github.com/Vineettyagi1/DATA-ANALYSIS-DASHBOARD/blob/main/dashboard%20project.xlsx">dataset</a>

 ## QUESTIONS (KPIs)
   The following metrics and insights are presented as key performance indicators "KPIs" that a business would want to track:
 - compare the sales and order using single chart.
 - which year got the highest sales and orders?
 - total sales.
 - Sales by Category and Sales by Sub-Category
 - Profit Gain Over Time
 - Monthly Sales Performance
 - Top 5 Customers by Profit
 - Sales by State/Location
 - Customer Count by Year

 - Dashboard intrection <a href="https://github.com/Vineettyagi1/DATA-ANALYSIS-DASHBOARD/blob/main/Screenshot%202026-01-30%20092720.png">Dashboard</a>

## Process
- Understanding the Business Problem: it's crucial to understand the business questions the dashboard aims to answer, such as total sales, profit changes over time, top customers, and category-wise sales
  differences
- Data Cleaning and Preparation
- Inserting Pivot Tables and Charts
- Creating the Dashboard Layout
- Inserting Slicers and Connecting Charts
- Customizing Charts and Dashboard Design
